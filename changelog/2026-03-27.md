# Changelog — March 27, 2026

## New Features

**Workflow run trigger info**

Workflow runs now display context about what triggered them. PR-triggered runs show the PR title as a clickable link to GitHub. Scheduled runs show when they were triggered.

**Workflow editing**

You can now edit existing workflows directly from the workflows list. Open a workflow and update its name, trigger, prompt, context repos, or schedule. You can also delete a workflow from the edit page.

## Updates

**Searchable repository pickers**

The trigger repo and context repos fields in the workflow form now have search. Type to filter the list instead of scrolling through all connected repositories.

## Bug Fixes

**Scheduled workflow cron display**

When editing a scheduled workflow, the schedule now correctly shows the preset (daily, weekly, monthly) instead of always falling back to "custom".

**Workflow error messages**

Failed workflow runs no longer surface internal error details. The run list now shows a clean "Workflow execution failed" message.
